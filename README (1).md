## Overview

This is a simple console-based Food Ordering System built in Java using
Object-Oriented Programming (OOP) concepts. Users can view a menu,
select food items with quantities, and generate a final bill.

## Project Files

-   FoodItem.java
-   Menu.java
-   OrderItem.java
-   Order.java
-   Main.java

## How to Run

1.  Compile all files: javac \*.java

2.  Run the program: java Main

## Features

-   Displays a menu of food items
-   Allows multiple item selection
-   Calculates total bill automatically
-   Uses classes and ArrayList
